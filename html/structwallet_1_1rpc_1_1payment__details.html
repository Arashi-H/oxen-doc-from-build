Detailed Description

Payment details struct

  • payment_id – Payment ID matching the input parameter.
  • tx_hash – Transaction hash used as the transaction ID.
  • amount – Amount for this payment.
  • block_height – Height of the block that first confirmed this payment.
  • unlock_time – Time (in block height) until this payment is safe to spend.
  • locked – If the payment is spendable or not
  • subaddr_index – Major & minor index, account and subaddress index respectively.
  • address – Address receiving the payment.
