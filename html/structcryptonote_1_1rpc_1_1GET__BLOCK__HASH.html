Detailed Description

Look up one or more blocks' hashes by their height.

Inputs:

  • heights array of block heights of which to look up the block hashes. Accepts at most 1000 heights per request.

Output values are pairs of heights as keys to block hashes as values:

  • status General RPC status string. "OK" means everything looks good.
  • height the current blockchain height of this node
  • <height> the block hash of the block with the given height. Note that each height key is the stringified integer value, e.g. "3456" rather than 3456.
