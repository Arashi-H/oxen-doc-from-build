Detailed Description

Specifies a "legacy" JSON RPC command, available via HTTP JSON at /whatever (in addition to json_rpc as "whatever"). When accessed via legacy mode the result is just the .result element of the JSON RPC response. (Only applies to the HTTP RPC interface, and does nothing if BINARY if specified).
