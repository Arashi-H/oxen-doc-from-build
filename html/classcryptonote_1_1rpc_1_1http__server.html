◆ shutdown()

void cryptonote::rpc::http_server::shutdown ( bool  join = false)

Closes the http server connection. Can safely be called multiple times, or to abort a startup if called before start().

Parameters
join- if true, wait for the proxy thread to exit. If false then joining will occur during destruction.

◆ start()

void cryptonote::rpc::http_server::start ( )

Starts the event loop in the thread handling http requests. Core must have been initialized and OxenMQ started. Will propagate an exception from the thread if startup fails.
