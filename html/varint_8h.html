Detailed Description

provides the implementation of varint's

Variable length integers ("varints") are serialized in little-endian order using 7 bits per byte where the most significant bit of each byte is a continuation bit (1 = more bytes coming, 0 = varint done) and the lower 7 bits are the next 7 significant bits. The varint sequence ends when there are no more significant bits to encode.
