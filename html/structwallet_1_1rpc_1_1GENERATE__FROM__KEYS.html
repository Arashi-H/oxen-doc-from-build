Detailed Description

Restore a wallet using the private spend key, view key and public address.

Inputs:

  • restore_height – (Optional: Default 0) Height in which to start scanning the blockchain for transactions into and out of this Wallet.
  • filename – Set the name of the wallet.
  • address – The public address of the wallet.
  • spendkey – The private spend key of the wallet
  • viewkey – The private view key of the wallet.
  • password – Set password for Wallet.
  • autosave_current – (Optional: Default true): If a pre-existing wallet is open, save to disk before opening the new wallet.

Outputs:

  • address – Restore a wallet using the seed words.
  • info – Restore a wallet using the seed words.
