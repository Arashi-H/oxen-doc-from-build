Detailed Description

Namespace for wallet RPC commands. Every RPC commands gets defined here and added to wallet_rpc_types list at the bottom of the file.

List of all supported rpc command structs to allow compile-time enumeration of all supported RPC types. Every type added above that has an RPC endpoint needs to be added here, and needs a core_rpc_server::invoke() overload that takes a <TYPE>::request and returns a <TYPE>::response. The <TYPE>::request has to be unique (for overload resolution); <TYPE>::response does not.
