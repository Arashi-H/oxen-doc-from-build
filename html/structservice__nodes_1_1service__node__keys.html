crypto::secret_key service_nodes::service_node_keys::key

The service node key pair used for registration-related data on the chain; is curve25519-based but with Monero-specific changes that make it useless for external tools supporting standard ed25519 or x25519 keys. TODO(oxen) - eventually drop this key and just do everything with the ed25519 key.

crypto::ed25519_secret_key service_nodes::service_node_keys::key_ed25519

A secondary SN key pair used for ancillary operations by tools (e.g. libsodium) that rely on standard cryptography keypair signatures.

◆ key_x25519

crypto::x25519_secret_key service_nodes::service_node_keys::key_x25519

A x25519 key computed from the ed25519 key, above, that is used for SN-to-SN encryption. (Unlike this above two keys this is not stored to disk; it is generated on the fly from the ed25519 key).
