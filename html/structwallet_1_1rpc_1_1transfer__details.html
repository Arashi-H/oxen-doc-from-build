Detailed Description

Transfer details struct

  • amount – Amount of this transfer.
  • spent – Indicates if this transfer has been spent.
  • global_index – The index into the global list of transactions grouped by amount in the Loki network.
  • tx_hash – Several incoming transfers may share the same hash if they were in the same transaction.
  • subaddr_index – Major & minor index, account and subaddress index respectively.
  • key_image – Key image for the incoming transfer's unspent output (empty unless verbose is true).
  • block_height – Block height the transfer occurred on
  • frozen – If the output has been intentionally frozen by the user, i.e. unspendable.
  • unlocked – If the TX is spendable yet
