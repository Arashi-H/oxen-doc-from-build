\doxysubsection{Detailed Description}
\subsubsection*{template$<$typename T, typename F = T, std\+::size\+\_\+t offset = 0$>$\newline
class lmdb\+::value\+\_\+iterator$<$ T, F, offset $>$}

An Input\+Iterator for a fixed-\/sized L\+M\+DB value at a specific key.


\begin{DoxyTemplParams}{Template Parameters}
{\em T} & The value type at the specific key. \\
\hline
{\em F} & The value type being returned when dereferenced. \\
\hline
{\em offset} & to {\ttfamily F} within {\ttfamily T}.\\
\hline
\end{DoxyTemplParams}
\begin{DoxyNote}{Note}
This meets requirements for an Input\+Iterator only. The iterator can only be incremented and dereferenced. All copies of an iterator share the same L\+M\+DB cursor, and therefore incrementing any copy will change the cursor state for all (incrementing an iterator will invalidate all prior copies of the iterator). Usage is identical to {\ttfamily std\+::istream\+\_\+iterator}. 
\end{DoxyNote}
