\doxysubsection{Detailed Description}
Prove a transaction by checking its signature.

Inputs\+:


\begin{DoxyItemize}
\item {\ttfamily txid} -- Transaction id.
\item {\ttfamily address} -- Destination public address of the transaction.
\item {\ttfamily message} -- (Optional) Should be the same message used in {\ttfamily get\+\_\+tx\+\_\+proof}.
\item {\ttfamily signature} -- Transaction signature to confirm.
\end{DoxyItemize}

Outputs\+:


\begin{DoxyItemize}
\item {\ttfamily good} -- States if the inputs proves the transaction.
\item {\ttfamily received} -- Amount of the transaction.
\item {\ttfamily in\+\_\+pool} -- States if the transaction is still in pool or has been added to a block.
\item {\ttfamily confirmations} -- Number of blocks mined after the one with the transaction. 
\end{DoxyItemize}
