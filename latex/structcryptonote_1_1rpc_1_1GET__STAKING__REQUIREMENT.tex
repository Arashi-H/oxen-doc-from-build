\doxysubsection{Detailed Description}
Get the required amount of Oxen to become a Service Node at the queried height. For devnet and testnet values, ensure the daemon is started with the {\ttfamily -\/-\/devnet} or {\ttfamily -\/-\/testnet} flags respectively.

Inputs\+:


\begin{DoxyItemize}
\item {\ttfamily height} The height to query the staking requirement for. 0 (or omitting) means current height.
\end{DoxyItemize}

Output values available from a public R\+PC endpoint\+:


\begin{DoxyItemize}
\item {\ttfamily status} generic R\+PC error code; \char`\"{}\+O\+K\char`\"{} means the request was successful.
\item {\ttfamily staking\+\_\+requirement} The staking requirement in Oxen, in atomic units.
\item {\ttfamily height} The height requested (or current height if 0 was requested) 
\end{DoxyItemize}
