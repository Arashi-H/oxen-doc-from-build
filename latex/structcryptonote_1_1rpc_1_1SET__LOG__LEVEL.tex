\doxysubsection{Detailed Description}
Set the daemon log level. By default, log level is set to {\ttfamily 0}. For more fine-\/tuned logging control set the set\+\_\+log\+\_\+categories command instead.

Inputs\+:
\begin{DoxyItemize}
\item {\ttfamily level} Daemon log level to set from {\ttfamily 0} (less verbose) to {\ttfamily 4} (most verbose)
\end{DoxyItemize}

Output values (requires a restricted/admin R\+PC endpoint)\+:


\begin{DoxyItemize}
\item {\ttfamily status} General R\+PC status string. {\ttfamily \char`\"{}\+O\+K\char`\"{}} means everything looks good. 
\end{DoxyItemize}
