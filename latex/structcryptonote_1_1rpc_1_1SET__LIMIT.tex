\doxysubsection{Detailed Description}

Inputs\+:

\item {\ttfamily limit\+\_\+down} Download limit in k\+Bytes per second. -\/1 means reset to default; 0 (or omitted) means don\textquotesingle{}t change the current limit

\end{DoxyItemize}
