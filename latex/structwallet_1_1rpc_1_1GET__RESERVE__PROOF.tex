\doxysubsection{Detailed Description}
Generate a signature to prove of an available amount in a wallet.

Inputs\+:


\begin{DoxyItemize}
\item {\ttfamily all} -- Proves all wallet balance to be disposable.
\item {\ttfamily account\+\_\+index} -- Specify the account from witch to prove reserve. (ignored if all is set to true)
\item {\ttfamily amount} -- Amount (in atomic units) to prove the account has for reserve. (ignored if all is set to true)
\item {\ttfamily message} -- (Optional) add a message to the signature to further authenticate the prooving process.
\end{DoxyItemize}

Outputs\+:


\begin{DoxyItemize}
\item {\ttfamily signature} -- Reserve signature. 
\end{DoxyItemize}
