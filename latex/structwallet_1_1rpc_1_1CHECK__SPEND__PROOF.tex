\doxysubsection{Detailed Description}
Prove a spend using a signature. Unlike proving a transaction, it does not requires the destination public address.

Inputs\+:


\begin{DoxyItemize}
\item {\ttfamily txid} -- Transaction id.
\item {\ttfamily message} -- (Optional) Should be the same message used in {\ttfamily get\+\_\+spend\+\_\+proof}.
\item {\ttfamily signature} -- Spend signature to confirm.
\end{DoxyItemize}

Outputs\+:


\begin{DoxyItemize}
\item {\ttfamily good} -- States if the inputs proves the spend. 
\end{DoxyItemize}
