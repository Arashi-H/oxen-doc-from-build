\doxysubsection{Detailed Description}
Returns a list of transfers, by default all transfer types are included. If all requested type fields are false, then all transfers will be queried.

Inputs\+:


\begin{DoxyItemize}
\item {\ttfamily in} -- (Optional) Include incoming transfers.
\item {\ttfamily out} -- (Optional) Include outgoing transfers.
\item {\ttfamily stake} -- (Optional) Include outgoing stakes.
\item {\ttfamily pending} -- (Optional) Include pending transfers.
\item {\ttfamily failed} -- (Optional) Include failed transfers.
\item {\ttfamily pool} -- (Optional) Include transfers from the daemon\textquotesingle{}s transaction pool.
\item {\ttfamily coinbase} -- (Optional) Include transfers from the daemon\textquotesingle{}s transaction pool.
\item {\ttfamily filter\+\_\+by\+\_\+height} -- (Optional) Filter transfers by block height.
\item {\ttfamily min\+\_\+height} -- (Optional) Minimum block height to scan for transfers, if filtering by height is enabled.
\item {\ttfamily max\+\_\+height} -- (Optional) Maximum block height to scan for transfers, if filtering by height is enabled (defaults to max block height).
\item {\ttfamily account\+\_\+index} -- (Optional) Index of the account to query for transfers. (defaults to 0)
\item {\ttfamily subaddr\+\_\+indices} -- (Optional) List of subaddress indices to query for transfers. (defaults to 0)
\item {\ttfamily all\+\_\+accounts} -- If true, return transfers for all accounts, subaddr\+\_\+indices and account\+\_\+index are ignored
\end{DoxyItemize}
