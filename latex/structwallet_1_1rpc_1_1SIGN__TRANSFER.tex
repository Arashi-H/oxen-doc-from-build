\doxysubsection{Detailed Description}
Sign a transaction created on a read-\/only wallet (in cold-\/signing process).

Inputs\+:


\begin{DoxyItemize}
\item {\ttfamily unsigned\+\_\+txset} -- Set of unsigned tx returned by \char`\"{}transfer\char`\"{} or \char`\"{}transfer\+\_\+split\char`\"{} methods.
\item {\ttfamily export\+\_\+raw} -- (Optional) If true, return the raw transaction data. (Defaults to false)
\item {\ttfamily get\+\_\+tx\+\_\+keys} -- (Optional) Return the transaction keys after sending.
\end{DoxyItemize}

Outputs\+:
