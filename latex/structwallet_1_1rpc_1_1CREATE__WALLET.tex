\doxysubsection{Detailed Description}
Create a new wallet. You need to have set the argument \char`\"{}\textquotesingle{}-\/-\/wallet-\/dir\char`\"{} when launching oxen-\/wallet-\/rpc to make this work.

Inputs\+:


\begin{DoxyItemize}
\item {\ttfamily filename} -- Set the wallet file name.
\item {\ttfamily password} -- (Optional) Set the password to protect the wallet.
\item {\ttfamily language} -- \mbox{\hyperlink{namespaceLanguage}{Language}} for your wallets\textquotesingle{} seed.
\item {\ttfamily hardware\+\_\+wallet} -- Create this wallet from a connected hardware wallet. ({\ttfamily language} will be ignored).
\item {\ttfamily device\+\_\+name} -- When {\ttfamily hardware} is true, this specifies the hardware wallet device type (currently supported\+: \char`\"{}\+Ledger\char`\"{}). If omitted \char`\"{}\+Ledger\char`\"{} is used.
\item {\ttfamily device\+\_\+label} -- (Optional) Custom label to write to a {\ttfamily wallet.\+hwdev.\+txt}. Can be empty; omit the parameter entirely to not write a .hwdev.\+txt file at all.
\end{DoxyItemize}

Outputs\+: None
