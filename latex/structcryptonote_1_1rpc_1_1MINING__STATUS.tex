\doxysubsection{Detailed Description}
Get the mining status of the daemon.

Inputs\+: none

Output values available from a restricted/admin R\+PC endpoint\+:


\begin{DoxyItemize}
\item {\ttfamily status} General R\+PC status string. {\ttfamily \char`\"{}\+O\+K\char`\"{}} means everything looks good.
\item {\ttfamily active} States if mining is enabled ({\ttfamily true}) or disabled ({\ttfamily false}).
\item {\ttfamily speed} Mining power in hashes per seconds.
\item {\ttfamily threads\+\_\+count} Number of running mining threads.
\item {\ttfamily address} Account address daemon is mining to. Empty if not mining.
\item {\ttfamily pow\+\_\+algorithm} Current hashing algorithm name
\item {\ttfamily block\+\_\+target} The expected time to solve per block, i.\+e. T\+A\+R\+G\+E\+T\+\_\+\+B\+L\+O\+C\+K\+\_\+\+T\+I\+ME
\item {\ttfamily block\+\_\+reward} Block reward for the current block being mined.
\item {\ttfamily difficulty} The difficulty for the current block being mined. 
\end{DoxyItemize}
