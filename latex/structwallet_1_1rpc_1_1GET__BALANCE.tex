\doxysubsection{Detailed Description}
Return the wallet\textquotesingle{}s balance.

Inputs\+:


\begin{DoxyItemize}
\item {\ttfamily account\+\_\+index} -- Return balance for this account.
\item {\ttfamily address\+\_\+indices} -- (Optional) Return balance detail for those subaddresses.
\item {\ttfamily all\+\_\+accounts} -- If true, return balance for all accounts, subaddr\+\_\+indices and account\+\_\+index are ignored
\item {\ttfamily strict} -- If true, only return the balance for transactions that have been spent and are not pending (i.\+e. excluding any transactions sitting in the TX pool)
\end{DoxyItemize}

\item {\ttfamily per\+\_\+subaddress} -- Balance information for each subaddress in an account.
