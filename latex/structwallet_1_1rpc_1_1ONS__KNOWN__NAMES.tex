\doxysubsection{Detailed Description}
Returns a list of known, plain-\/text O\+NS names along with record details for names that this wallet knows about. This can optionally decrypt the O\+NS value as well, or else just return the encrypted value.

Inputs\+:


\begin{DoxyItemize}
\item {\ttfamily decrypt} -- If true (default false) then also decrypt and include the {\ttfamily value} field
\item {\ttfamily include\+\_\+expired} -- If true (default false) then also include expired records
\end{DoxyItemize}

Outputs\+:


\begin{DoxyItemize}
\item {\ttfamily known\+\_\+names} -- List of records known to this wallet
\begin{DoxyItemize}
\item {\ttfamily type} -- The mapping type, \char`\"{}session\char`\"{} or \char`\"{}lokinet\char`\"{}.
\item {\ttfamily hashed} -- The hashed name (in base64)
\item {\ttfamily name} -- The plaintext name
\item {\ttfamily owner} -- The public key that purchased the Loki Name Service entry.
\item {\ttfamily backup\+\_\+owner} -- The backup public key or wallet that the owner specified when purchasing the Loki Name Service entry. Omitted if no backup owner.
\item {\ttfamily encrypted\+\_\+value} -- The encrypted value that the name maps to, in hex.
\item {\ttfamily value} -- Decrypted value that that name maps to. Only provided if {\ttfamily decrypt\+: true} was specified in the request.
\item {\ttfamily update\+\_\+height} -- The last height that this Loki Name Service entry was updated on the Blockchain.
\item {\ttfamily expiration\+\_\+height} -- For records that expire, this will be set to the expiration block height.
\item {\ttfamily expired} -- Indicates whether the record has expired. Only included in the response if \char`\"{}include\+\_\+expired\char`\"{} is specified in the request.
\item {\ttfamily txid} -- The txid of the mapping\textquotesingle{}s most recent update or purchase. 
\end{DoxyItemize}
\end{DoxyItemize}
