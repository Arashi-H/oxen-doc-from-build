\doxysubsection{Detailed Description}
Similar to get\+\_\+block\+\_\+header\+\_\+by\+\_\+height above, but for a range of blocks. This method includes a starting block height and an ending block height as parameters to retrieve basic information about the range of blocks.

Inputs\+:


\begin{DoxyItemize}
\item {\ttfamily start\+\_\+height} The starting block\textquotesingle{}s height.
\item {\ttfamily end\+\_\+height} The ending block\textquotesingle{}s height.
\item {\ttfamily fill\+\_\+pow\+\_\+hash} Tell the daemon if it should fill out pow\+\_\+hash field.
\item {\ttfamily get\+\_\+tx\+\_\+hashes} If true (default false) then include the hashes of non-\/coinbase transactions
\end{DoxyItemize}

Output values available from a public R\+PC endpoint\+:


\begin{DoxyItemize}
\item {\ttfamily status} General R\+PC status string. {\ttfamily \char`\"{}\+O\+K\char`\"{}} means everything looks good.
\item {\ttfamily headers} Array of \mbox{\hyperlink{structcryptonote_1_1block__header}{block\+\_\+header}} (a structure containing block header information. See get\+\_\+last\+\_\+block\+\_\+header). 
\end{DoxyItemize}
