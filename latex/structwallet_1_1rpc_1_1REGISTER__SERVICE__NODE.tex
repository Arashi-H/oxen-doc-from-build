Register Service Node.

Inputs\+:


\begin{DoxyItemize}
\item {\ttfamily register\+\_\+service\+\_\+node\+\_\+str} -- String supplied by the prepare\+\_\+registration command.
\item {\ttfamily get\+\_\+tx\+\_\+key} -- (Optional) Return the transaction key after sending.
