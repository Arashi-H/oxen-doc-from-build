\doxysubsection{Detailed Description}
Return the wallet\textquotesingle{}s addresses for an account. Optionally filter for specific set of subaddresses.

Inputs\+:


\begin{DoxyItemize}
\item {\ttfamily account\+\_\+index} -- Get the wallet addresses for the specified account.
\item {\ttfamily address\+\_\+index} -- (Optional) List of subaddresses to return from the aforementioned account.
\end{DoxyItemize}

Outputs\+:


\begin{DoxyItemize}
\item {\ttfamily address} -- (Deprecated) Remains to be compatible with older R\+PC format
\item {\ttfamily addresses} -- Addresses informations.
\begin{DoxyItemize}
\item {\ttfamily address} -- The (sub)address string.
\item {\ttfamily label} -- Label of the (sub)address.
\item {\ttfamily address\+\_\+index} -- Index of the subaddress
\item {\ttfamily used} -- True if the (sub)address has received funds before. 
\end{DoxyItemize}
\end{DoxyItemize}
