\doxysubsection{Detailed Description}
Ban another node by IP.

Inputs\+:
\begin{DoxyItemize}
\item {\ttfamily host} Banned host (IP in A.\+B.\+C.\+D form).
\item {\ttfamily ip} Banned IP address, in Int format.
\item {\ttfamily seconds} Local Unix time that IP is banned until, or Number of seconds to ban node
\item {\ttfamily ban} Set true to ban.
\end{DoxyItemize}
