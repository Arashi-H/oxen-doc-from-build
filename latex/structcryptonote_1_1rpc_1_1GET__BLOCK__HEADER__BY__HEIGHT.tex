\doxysubsection{Detailed Description}
Similar to get\+\_\+block\+\_\+header\+\_\+by\+\_\+hash above, this method includes a block\textquotesingle{}s height as an input parameter to retrieve basic information about the block.

Inputs\+:


\begin{DoxyItemize}
\item {\ttfamily height} A block height to look up; returned in {\ttfamily \mbox{\hyperlink{structcryptonote_1_1block__header}{block\+\_\+header}}}
\item {\ttfamily heights} Block heights to retrieve; returned in {\ttfamily block\+\_\+headers}
\item {\ttfamily fill\+\_\+pow\+\_\+hash} Tell the daemon if it should fill out pow\+\_\+hash field.
\item {\ttfamily get\+\_\+tx\+\_\+hashes} If true (default false) then include the hashes of non-\/coinbase transactions
\end{DoxyItemize}

Output values available from a public R\+PC endpoint\+:


\begin{DoxyItemize}
\item {\ttfamily status} General R\+PC status string. {\ttfamily \char`\"{}\+O\+K\char`\"{}} means everything looks good.
\item {\ttfamily \mbox{\hyperlink{structcryptonote_1_1block__header}{block\+\_\+header}}} Block header information for the requested {\ttfamily height} block
\item {\ttfamily block\+\_\+headers} Block header information for the requested {\ttfamily heights} blocks 
\end{DoxyItemize}
