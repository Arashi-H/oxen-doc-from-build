\doxysubsection{Detailed Description}
Look up one or more blocks\textquotesingle{} hashes by their height.

Inputs\+:
\begin{DoxyItemize}
\item heights array of block heights of which to look up the block hashes. Accepts at most 1000 heights per request.
\end{DoxyItemize}

Output values are pairs of heights as keys to block hashes as values\+:
\begin{DoxyItemize}
\item {\ttfamily status} General R\+PC status string. {\ttfamily \char`\"{}\+O\+K\char`\"{}} means everything looks good.
\item {\ttfamily height} the current blockchain height of this node
\item {\ttfamily } $<$height$>$ the block hash of the block with the given height. Note that each height key is the stringified integer value, e.\+g. \char`\"{}3456\char`\"{} rather than 3456. 
\end{DoxyItemize}
