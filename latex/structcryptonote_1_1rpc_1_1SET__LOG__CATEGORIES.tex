\doxysubsection{Detailed Description}
Set the daemon log categories. Categories are represented as a comma separated list of {\ttfamily $<$Category$>$\+:$<$level$>$} (similarly to syslog standard {\ttfamily $<$Facility$>$\+:$<$Severity-\/level$>$}), where\+: Category is one of the following\+: $\ast$ (all facilities), default, net, net.\+http, net.\+p2p, logging, net.\+trottle, blockchain.\+db, blockchain.\+db.\+lmdb, bcutil, checkpoints, net.\+dns, net.\+dl, i18n, perf,stacktrace, updates, account, cn ,difficulty, hardfork, miner, blockchain, txpool, cn.\+block\+\_\+queue, net.\+cn, daemon, debugtools.\+deserialize, debugtools.\+objectsizes, device.\+ledger, wallet.\+gen\+\_\+multisig, multisig, bulletproofs, ringct, daemon.\+rpc, wallet.\+simplewallet, Wallet\+A\+PI, wallet.\+ringdb, wallet.\+wallet2, \mbox{\hyperlink{namespacewallet_1_1rpc}{wallet.\+rpc}}, tests.\+core.

Level is one of the following\+: F\+A\+T\+AL -\/ higher level, E\+R\+R\+OR, W\+A\+R\+N\+I\+NG, I\+N\+FO, D\+E\+B\+UG, T\+R\+A\+CE.

Lower level A level automatically includes higher level. By default, categories are set to\+: {\ttfamily $\ast$\+:W\+A\+R\+N\+I\+NG,net\+:F\+A\+T\+AL,net.\+p2p\+:F\+A\+T\+AL,net.\+cn\+:F\+A\+T\+AL,global\+:I\+N\+FO,verify\+:F\+A\+T\+AL,stacktrace\+:I\+N\+FO,logging\+:I\+N\+FO,msgwriter\+:I\+N\+FO} Setting the categories to \char`\"{}\char`\"{} prevents any log output.

You can append to the current the log level for updating just one or more categories while leaving other log levels unchanged by specifying one or more \char`\"{}$<$category$>$\+:$<$level$>$\char`\"{} pairs preceded by a \char`\"{}+\char`\"{}, for example \char`\"{}+difficulty\+:\+D\+E\+B\+U\+G,net\+:\+W\+A\+R\+N\+I\+N\+G\char`\"{}.

Inputs\+:


\begin{DoxyItemize}
\item {\ttfamily categories} Optional, daemon log categores to enable
\end{DoxyItemize}

Output values (requires a restricted/admin R\+PC endpoint)\+:


\begin{DoxyItemize}
\item {\ttfamily status} General R\+PC status string. {\ttfamily \char`\"{}\+O\+K\char`\"{}} means everything looks good.
\item {\ttfamily categories} Daemon log enabled categories 
\end{DoxyItemize}
