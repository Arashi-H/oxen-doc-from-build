\doxysubsection{Detailed Description}
\subsubsection*{template$<$typename K, typename V$>$\newline
class lmdb\+::key\+\_\+iterator$<$ K, V $>$}

An Input\+Iterator for a fixed-\/sized L\+M\+DB key and value. {\ttfamily operator++} iterates over keys.


\begin{DoxyTemplParams}{Template Parameters}
{\em K} & Key type in database records. \\
\hline
{\em V} & Value type in database records.\\
\hline
\end{DoxyTemplParams}
\begin{DoxyNote}{Note}
This meets requirements for an Input\+Iterator only. The iterator can only be incremented and dereferenced. All copies of an iterator share the same L\+M\+DB cursor, and therefore incrementing any copy will change the cursor state for all (incrementing an iterator will invalidate all prior copies of the iterator). Usage is identical to {\ttfamily std\+::istream\+\_\+iterator}. 
\end{DoxyNote}
