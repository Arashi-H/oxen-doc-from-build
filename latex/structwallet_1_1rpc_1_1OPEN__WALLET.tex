\doxysubsection{Detailed Description}
Open a wallet. You need to have set the argument \char`\"{}-\/-\/wallet-\/dir\char`\"{} when launching oxen-\/wallet-\/rpc to make this work. The wallet rpc executable may only open wallet files within the same directory as wallet-\/dir, otherwise use the \char`\"{}-\/-\/wallet-\/file\char`\"{} flag to open specific wallets.

Inputs\+:


\begin{DoxyItemize}
\item {\ttfamily filename} -- \mbox{\hyperlink{classwallet_1_1Wallet}{Wallet}} name stored in \char`\"{}-\/-\/wallet-\/dir\char`\"{}.
\item {\ttfamily password} -- The wallet password, set as \char`\"{}\char`\"{} if there\textquotesingle{}s no password
\item {\ttfamily autosave\+\_\+current} -- (Optional\+: Default true)\+: If a pre-\/existing wallet is open, save to disk before opening the new wallet.
\end{DoxyItemize}

Outputs\+: None
