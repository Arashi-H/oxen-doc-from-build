\doxysubsection{Detailed Description}
Query hardcoded/service node checkpoints stored for the blockchain. Omit all arguments to retrieve the latest \char`\"{}count\char`\"{} checkpoints.

Inputs\+:


\begin{DoxyItemize}
\item {\ttfamily start\+\_\+height} Optional\+: Get the first count checkpoints starting from this height. Specify both start and end to get the checkpoints inbetween.
\item {\ttfamily end\+\_\+height} Optional\+: Get the first count checkpoints before end height. Specify both start and end to get the checkpoints inbetween.
\item {\ttfamily count} Optional\+: Number of checkpoints to query.
\end{DoxyItemize}

Output values available from a public R\+PC endpoint\+:


\begin{DoxyItemize}
\item {\ttfamily status} generic R\+PC error code; \char`\"{}\+O\+K\char`\"{} means the request was successful.
\item {\ttfamily checkpoints} Array of requested checkpoints 
\end{DoxyItemize}
