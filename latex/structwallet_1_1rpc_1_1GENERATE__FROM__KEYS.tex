\doxysubsection{Detailed Description}
Restore a wallet using the private spend key, view key and public address.

Inputs\+:


\begin{DoxyItemize}
\item {\ttfamily restore\+\_\+height} -- (Optional\+: Default 0) Height in which to start scanning the blockchain for transactions into and out of this \mbox{\hyperlink{classwallet_1_1Wallet}{Wallet}}.
\item {\ttfamily filename} -- Set the name of the wallet.
\item {\ttfamily address} -- The public address of the wallet.
\item {\ttfamily spendkey} -- The private spend key of the wallet
\item {\ttfamily viewkey} -- The private view key of the wallet.
\item {\ttfamily password} -- Set password for \mbox{\hyperlink{classwallet_1_1Wallet}{Wallet}}.
\item {\ttfamily autosave\+\_\+current} -- (Optional\+: Default true)\+: If a pre-\/existing wallet is open, save to disk before opening the new wallet.
