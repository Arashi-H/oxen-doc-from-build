Inputs\+: None

\item {\ttfamily chains} Array of Chains. Each element is contains the following keys\+:
\begin{DoxyItemize}
\item {\ttfamily block\+\_\+hash} The block hash of the first diverging block of this alternative chain.
\item {\ttfamily height} The block height of the first diverging block of this alternative chain.
\item {\ttfamily length} The length in blocks of this alternative chain, after divergence.
\item {\ttfamily difficulty} The cumulative difficulty of all blocks in the alternative chain.
