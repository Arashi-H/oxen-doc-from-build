\doxysubsection{Detailed Description}
Get all accounts for a wallet. Optionally filter accounts by tag.

Inputs\+:


\begin{DoxyItemize}
\item {\ttfamily tag} -- (Optional) Tag for filtering accounts. All accounts if empty, otherwise those accounts with this tag
\item {\ttfamily strict\+\_\+balances} -- If true, only return the balance for transactions that have been spent and are not pending (i.\+e. excluding any transactions sitting in the TX pool)
\end{DoxyItemize}

Outputs\+:


\begin{DoxyItemize}
\item {\ttfamily total\+\_\+balance} -- Total balance of the selected accounts (locked or unlocked).
\item {\ttfamily total\+\_\+unlocked\+\_\+balance} -- Total unlocked balance of the selected accounts.
\item {\ttfamily subaddress\+\_\+accounts} -- Account information.
\begin{DoxyItemize}
\item {\ttfamily account\+\_\+index} -- Index of the account.
\item {\ttfamily base\+\_\+address} -- The first address of the account (i.\+e. the primary address).
\item {\ttfamily balance} -- Balance of the account (locked or unlocked).
\item {\ttfamily unlocked\+\_\+balance} -- Unlocked balance for the account.
\item {\ttfamily label} -- (Optional) Label of the account.
\item {\ttfamily tag} -- (Optional) Tag for filtering accounts. 
\end{DoxyItemize}
