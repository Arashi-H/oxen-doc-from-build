\doxysubsection{Detailed Description}
Create a new address for an account. Optionally, label the new address.

Inputs\+:


\begin{DoxyItemize}
\item {\ttfamily account\+\_\+index} -- Create a new subaddress for this account.
\item {\ttfamily label} -- (Optional) Label for the new subaddress.
\item {\ttfamily count} -- Number of addresses to create, defaults to 1.
\end{DoxyItemize}

Outputs\+:


\begin{DoxyItemize}
\item {\ttfamily address} -- The newly requested address.
\item {\ttfamily address\+\_\+index} -- Index of the new address in the requested account index.
\item {\ttfamily addresses} -- The new addresses, if more than 1 is requested
\item {\ttfamily address\+\_\+indices} -- The new addresses indicies if more than 1 is requested 
\end{DoxyItemize}
