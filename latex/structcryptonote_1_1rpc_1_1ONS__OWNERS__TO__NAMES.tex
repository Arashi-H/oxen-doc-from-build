\doxysubsection{Detailed Description}
Get all the name mappings for the queried owner. The owner can be either a ed25519 public key or Monero style public key; by default purchases are owned by the spend public key of the purchasing wallet.

Inputs\+:


\begin{DoxyItemize}
\item {\ttfamily entries} List of owner\textquotesingle{}s public keys to find all Oxen Name Service entries for.
\item {\ttfamily include\+\_\+expired} Optional\+: if provided and true, include entries in the results even if they are expired
\end{DoxyItemize}

Output values available from a public R\+PC endpoint\+:


\begin{DoxyItemize}
\item {\ttfamily status} Generic R\+PC error code. \char`\"{}\+O\+K\char`\"{} is the success value.
\item {\ttfamily entries} List of O\+NS names. Each element is structured as follows\+:
\begin{DoxyItemize}
\item {\ttfamily request\+\_\+index} (Deprecated) The index in request\textquotesingle{}s {\ttfamily entries} array that was resolved via Loki Name Service.
\item {\ttfamily type} The category the Loki Name Service entry belongs to; currently 0 for Session, 1 for \mbox{\hyperlink{namespaceWallet}{Wallet}} and 2 for Lokinet.
\item {\ttfamily name\+\_\+hash} The hash of the name that the owner purchased via Loki Name Service in base64
\item {\ttfamily owner} The backup public key specified by the owner that purchased the Loki Name Service entry.
\item {\ttfamily backup\+\_\+owner} The backup public key specified by the owner that purchased the Loki Name Service entry. Omitted if no backup owner.
\item {\ttfamily encrypted\+\_\+value} The encrypted value that the name maps to, in hex. This value is encrypted using the name (not the hash) as the secret.
\item {\ttfamily update\+\_\+height} The last height that this Loki Name Service entry was updated on the \mbox{\hyperlink{classcryptonote_1_1Blockchain}{Blockchain}}.
\item {\ttfamily expiration\+\_\+height} For records that expire, this will be set to the expiration block height.
\item {\ttfamily txid} The txid of the mapping\textquotesingle{}s most recent update or purchase. 
\end{DoxyItemize}
\end{DoxyItemize}
