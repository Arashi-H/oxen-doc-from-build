\doxysubsection{Detailed Description}
Get a histogram of output amounts. For all amounts (possibly filtered by parameters), gives the number of outputs on the chain for that amount. Ring\+CT outputs counts as 0 amount.

Inputs\+:


\begin{DoxyItemize}
\item {\ttfamily amounts} list of amounts in Atomic Units.
\item {\ttfamily min\+\_\+count} The minimum amounts you are requesting.
\item {\ttfamily max\+\_\+count} The maximum amounts you are requesting.
\item {\ttfamily unlocked} Look for locked only.
\item {\ttfamily recent\+\_\+cutoff} 
\end{DoxyItemize}

Output values available from a public R\+PC endpoint\+:


\begin{DoxyItemize}
\item {\ttfamily status} General R\+PC status string. {\ttfamily \char`\"{}\+O\+K\char`\"{}} means everything looks good.
\item {\ttfamily histogram} List of histogram entries. Each element is structured as follows\+:
\begin{DoxyItemize}
\item {\ttfamily uint64\+\_\+t} amount Output amount in atomic units.
\item {\ttfamily uint64\+\_\+t} total\+\_\+instances
\item {\ttfamily uint64\+\_\+t} unlocked\+\_\+instances
\item {\ttfamily uint64\+\_\+t} recent\+\_\+instances 
\end{DoxyItemize}
\end{DoxyItemize}
