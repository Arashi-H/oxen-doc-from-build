\doxysubsection{Detailed Description}
Get the coinbase amount and the fees amount for n last blocks starting at particular height.

Inputs\+:


\begin{DoxyItemize}
\item {\ttfamily height} Block height from which getting the amounts.
\item {\ttfamily count} Number of blocks to include in the sum.
\end{DoxyItemize}

Output values available from a restricted/admin R\+PC endpoint\+:


\begin{DoxyItemize}
\item {\ttfamily status} General R\+PC status string. {\ttfamily \char`\"{}\+O\+K\char`\"{}} means everything looks good.
\item {\ttfamily emission\+\_\+amount} Amount of coinbase reward in atomic units.
\item {\ttfamily fee\+\_\+amount} Amount of fees in atomic units.
\item {\ttfamily burn\+\_\+amount} Amount of burnt oxen. 
\end{DoxyItemize}
