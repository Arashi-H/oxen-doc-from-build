{\footnotesize\ttfamily \mbox{\hyperlink{stdbool_8h_a1062901a7428fdd9c7f180f5e01ea056}{bool}} cryptonote\+::levin\+::notify\+::send\+\_\+txs (\begin{DoxyParamCaption}\item[{std\+::vector$<$ std\+::string $>$}]{txs,  }\item[{const boost\+::uuids\+::uuid \&}]{source,  }\item[{\mbox{\hyperlink{stdbool_8h_a1062901a7428fdd9c7f180f5e01ea056}{bool}}}]{pad\+\_\+txs }\end{DoxyParamCaption})}

Send txs using {\ttfamily \mbox{\hyperlink{cryptonote__protocol__defs_8h}{cryptonote\+\_\+protocol\+\_\+defs.\+h}}} payload format wrapped in a levin header. The message will be sent in a \char`\"{}discreet\char`\"{} manner if enabled -\/ if {\ttfamily !noise.empty()} then the {\ttfamily command}/{\ttfamily payload} will be queued to send at the next available noise interval. Otherwise, a standard Monero flood notification will be used.

\begin{DoxyNote}{Note}
Eventually Dandelion++ stem sending will be used here when enabled.
\end{DoxyNote}
