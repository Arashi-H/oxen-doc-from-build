\doxysubsection{Detailed Description}
Queries whether outputs have been spent using the key image associated with the output.

Inputs\+:


\begin{DoxyItemize}
\item {\ttfamily key\+\_\+images} list of key images to check. For json requests these must be hex or base64-\/encoded; for bt-\/requests they can be hex/base64 or raw bytes.
\end{DoxyItemize}

Outputs


\begin{DoxyItemize}
\item {\ttfamily status} General R\+PC status string. {\ttfamily \char`\"{}\+O\+K\char`\"{}} means everything looks good.
\item {\ttfamily spent\+\_\+status} array of status codes returned in the same order as the {\ttfamily key\+\_\+images} input. Each value is one of\+:
\begin{DoxyItemize}
\item {\ttfamily 0} the key image is unspent
\item {\ttfamily 1} the key image is spent in a mined block
\item {\ttfamily 2} the key image is spent in a transaction currently in the mempool 
\end{DoxyItemize}
\end{DoxyItemize}
