\doxysubsection{Detailed Description}
Reports service node peer status (success/fail) from lokinet and storage server.

Inputs\+:


\begin{DoxyItemize}
\item {\ttfamily type} test type; currently supported are\+: \char`\"{}storage\char`\"{} and \char`\"{}lokinet\char`\"{} for storage server and lokinet tests, respectively.
\item {\ttfamily pubkey} service node pubkey
\item {\ttfamily passed} whether node is passing the test
\end{DoxyItemize}

Output values available from a private/admin R\+PC endpoint\+:


\begin{DoxyItemize}
\item {\ttfamily status} Generic R\+PC error code. \char`\"{}\+O\+K\char`\"{} is the success value.
