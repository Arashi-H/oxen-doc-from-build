\doxysubsection{Detailed Description}
Start mining on the daemon

Inputs\+:


\begin{DoxyItemize}
\item {\ttfamily miner\+\_\+address} Account address to mine to.
\item {\ttfamily threads\+\_\+count} Number of mining threads to run. Defaults to 1 thread if omitted or 0.
\item {\ttfamily num\+\_\+blocks} Mine until the blockchain has this many new blocks, then stop (no limit if 0, the default).
\item {\ttfamily slow\+\_\+mining} Do slow mining (i.\+e. don\textquotesingle{}t allocate RandomX cache); primarily intended for testing.
\end{DoxyItemize}
