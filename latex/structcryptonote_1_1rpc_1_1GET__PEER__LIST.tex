\doxysubsection{Detailed Description}
Get the list of current network peers known to this node.

Inputs\+: none

Output values (requires a restricted/admin R\+PC endpoint)\+:


\begin{DoxyItemize}
\item {\ttfamily status} General R\+PC status string. {\ttfamily \char`\"{}\+O\+K\char`\"{}} means everything looks good.
\item {\ttfamily white\+\_\+list} list of \char`\"{}whitelist\char`\"{} peers (see below), that is, peers that were recorded reachable the last time this node connected to them. Peers that are unreachable or not synchronized with the network are moved to the graylist.
\item {\ttfamily gray\+\_\+list} list of peers (see below) that this node knows of but has not (recently) tried to connect to.
\end{DoxyItemize}

Each peer list is an array of dicts containing the following fields\+:
\begin{DoxyItemize}
\item {\ttfamily id} a unique integer locally identifying the peer
\item {\ttfamily host} the peer\textquotesingle{}s IP address (as a string)
\item {\ttfamily port} the port on which the peer is reachable
\item {\ttfamily last\+\_\+seen} unix timestamp when this node last connected to the peer. Will be omitted if never connected (e.\+g. for a peer we received from another node but haven\textquotesingle{}t yet tried). 
\end{DoxyItemize}
